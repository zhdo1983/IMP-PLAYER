一、欢迎使用

感谢您选择我们的ESP32网络收音机！这是一款集网络电台、本地MP3播放、时钟显示、娱乐游戏于一体的多功能设备。它操作简单，音质出色，能为您的生活带来更多乐趣。

主要功能概览：

  • 海量网络电台： 轻松收听全球广播，支持MP3/AAC格式。
  • SD卡音乐播放： 插入TF卡即可播放您喜爱的MP3音乐。
  • 清晰OLED显示： 128x64像素屏幕，中文菜单，信息一目了然。
  • 旋转编码器： 方便快捷地调节音量、选择菜单。
  • 智能WiFi配置： 首次使用或更换网络时，通过手机即可轻松设置。
  • 在线固件升级： 随时获取最新功能和性能优化。
  • 电池电量显示： 实时了解设备电量，方便充电。
  • NTP网络时钟： 自动同步准确时间。
  • 趣味PONG游戏： 享受休闲时光。

四、首次使用

4.1 连接电源与开机

1. 将USB充电线连接到设备的USB接口，另一端连接到5V USB电源适配器（例如手机充电器）或电脑USB端口。

2. 设备将自动开机，并显示启动动画。

4.2 WiFi配置（首次使用或更换网络）

首次开机或设备无法连接到已保存的WiFi时，它会自动进入配网模式。此时OLED屏幕会显示相关提示。

  1. 连接设备热点：

    用您的手机或电脑搜索WiFi热点，找到名为 ESP32_Radio 的网络并连接（无需密码）。

    [图片占位符] 手机连接ESP32_Radio热点界面

    请在此处替换为手机连接名为“ESP32_Radio”WiFi热点的界面截图。

  2. 打开配置页面：

    连接成功后，打开手机或电脑的浏览器，访问地址：http://192.168.3.3。

    您将看到一个简单的配置页面。

    [图片占位符] WiFi配置页面

    请在此处替换为浏览器中访问192.168.3.3时显示的WiFi配置页面截图。

  3. 输入WiFi信息：

    在页面上输入您家或办公室的WiFi名称（SSID）和密码。

    您可以添加最多5个常用的WiFi网络。

  4. 保存并重启：

    点击页面底部的“保存并重启”按钮。

    设备会自动重启并尝试连接您配置的WiFi网络。连接成功后，它会尝试播放默认电台。

4.3 TF卡准备与插入（如需播放本地音乐）

  1. 将TF卡（建议使用Class 10或更高速度，最大支持32GB）插入电脑。
  2. 格式化： 如果TF卡不是FAT32格式，请将其格式化为FAT32。
  3. 复制音乐： 将您喜欢的MP3音乐文件复制到TF卡中（支持子文件夹）。
  4. 插入设备： 将准备好的TF卡轻轻插入设备的TF卡槽中。

五、功能操作指南

5.1 音量调节

  • 在任何播放界面，旋转编码器即可调节音量大小。
  • 顺时针旋转：增大音量。
  • 逆时针旋转：减小音量。

5.2 主菜单

  • 在播放界面，短按旋转编码器按钮，即可进入主菜单。
  • 在主菜单中，通过旋转编码器来选择不同的功能。
  • 短按编码器按钮，确认进入所选功能。
  • 如果10秒内没有任何操作，菜单会自动返回到播放界面。

主菜单功能包括：

  1. 选择电台： 播放网络电台。

  6. 恢复出厂设置： 清除所有用户设置，恢复默认。
  7. 内存分配： （此项为技术信息，普通用户可忽略）查看设备内存使用情况。

七、常见问题解答

Q1：设备无法连接WiFi怎么办？
A：
  1. 检查您输入的WiFi名称（SSID）和密码是否正确，注意大小写。
  2. 尝试将设备靠近路由器，或检查WiFi信号强度。
  3. 尝试重启设备，然后重新进行WiFi配置。
  4. 确保您的WiFi是2.4GHz频段，目前不支持5GHz WiFi。
Q2：播放音乐没有声音？
A：
  1. 检查音量是否调到最低，尝试旋转编码器增大音量。
  2. 检查扬声器是否正确连接，是否有松动。
  3. 确保MAX98357A功放模块已正常供电。
Q3：SD卡中的音乐无法播放？
A：
  1. 确保TF卡已格式化为FAT32文件系统。
  2. 确保TF卡中存放的是MP3格式的音乐文件。
  3. 检查TF卡是否插紧或损坏。
  4. 尝试重新插入TF卡，然后再次进入SD卡播放器。
Q4：设备播放时卡顿，或者频繁重启？
A：
  1. 检查网络连接是否稳定，如果播放网络电台，可能是网络速度慢导致。
  2. 如果播放SD卡音乐，可能是TF卡读写速度不够（建议Class 10或更高）。
  3. 如果TF卡中音乐文件过多（超过500首），可能会影响设备性能，尝试减少歌曲数量。
  4. 尝试重启设备，释放内存。
Q5：如何获取最新固件？
A： 您可以通过设备自带的OTA固件升级功能（参见5.9节）在线更新，无需连接电脑。

八、安全注意事项

为了您的安全和设备正常运行，请务必遵守以下注意事项：

  1. 电源： 请使用符合安全标准的5V USB电源适配器。切勿使用劣质电源或连接电压不符的电源。
  2. 电池： 如果设备使用电池供电，请确保使用原装或符合标准的锂电池，并注意电池的正负极连接。避免电池过充、过放、短路或刺穿。
  3. 环境： 避免在潮湿、高温、多尘或腐蚀性气体环境中使用设备。
  4. 操作： 请勿私自拆卸、改装设备。避免强力撞击或摔落。
  5. 网络： 谨慎连接未知或不安全的WiFi网络。
